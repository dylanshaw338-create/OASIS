# OASIS v0.3 版本更新日志 (Release Notes)

## 🌟 核心新特性 (New Features)

### 1. 史诗级宇宙 UI 与沉浸光影重构 (Cinematic Space UI)
*   **深渊黑曜石质感**：重构了顶栏、底部导航与弹窗等所有玻璃态组件，采用深邃的 `#121620` 悬浮底板配合极细的边缘高光与深邃投影，打造物理级厚重感。
*   **多层次星空系统**：
    *   **星云呼吸**：底层加入了长达 120 秒循环周期的巨型暗紫/深蓝星云漂移。
    *   **流星划过**：引入了低频触发的流星系统，带有逼真的亮核与渐变长尾迹，极具科幻感。
    *   **星际航行**：180 颗星星加入了带有深度错觉（Depth Factor）的微弱定向漂移，配合极其缓慢的明暗呼吸，模拟置身太空舱的航行体验。
*   **电影级开场封面 (Splash Screen)**：
    *   主标题升级为超细字体、超大字距的 `FUTURE HCI`。
    *   重构入场动效：从高斯模糊中渐隐浮现，并带有赛博冰蓝与虚空紫的双重霓虹光晕点亮效果。
    *   加入带发光节点的神经链接展开动画及赛博扫描线网格底纹。

### 2. AI 神经链接面板大改版 (AI Panel Refactor)
*   **多会话管理 (Sessions)**：
    *   新增历史对话侧边栏，支持无缝切换历史对话。
    *   自动根据用户首条输入生成会话标题。
    *   支持旧版单线程数据的安全向下兼容与自动迁移。
*   **ChatGPT 范式流式排版**：
    *   彻底抛弃左右对称的彩色气泡，采用统一左对齐的流式文档布局。
    *   引入专属身份头像（用户 / AI 节点），极大提升深度阅读的专业感与沉浸感。
*   **微信级复合输入底板**：
    *   将输入框移至底部，重构为支持多行输入 (`textarea`) 的极简深灰底板。
    *   内置工具占位符与动态点亮的“发送”按钮。
*   **精细化上下文控制**：
    *   配置页新增「融合本地思想」开关，允许自由决定是否将近期笔记作为 Prompt 注入，节省 Token 消耗。
    *   新增**批量删除管理模式 (Batch Delete)**，支持多选 Checkbox 一键清空冗余对话。
    *   单条消息 Hover 时提供克制的复制/删除极简图标。

### 3. 交互与细节打磨 (UX & Details)
*   **焦点劫持修复**：彻底修复了因自动保存 (`autoSave`) 导致的输入标题时焦点被抢夺、光标乱跳的恶性 Bug。
*   **列表排序防抖**：引入严格的脏检查（Dirty Check），修复了在思想区单纯点击笔记会导致时间戳更新及列表乱跳的问题。
*   **系统退出弹窗重制**：`ExitDialog` 升级为暗黑预警风格，取消/退出按钮采用幽灵与暗红悬浮交互，强化“断开神经连接”的仪式感。
