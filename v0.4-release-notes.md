# Future HCI v0.4.0 Release Notes

## 🎉 新特性 (New Features)
- **知识库模块 (Papers)**：全新推出本地文献管理界面，支持多栏布局与 PDF 文件的导入、管理。
- **内嵌 PDF 阅读器**：突破 Chromium 本地文件安全限制，实现应用内原生、无缝的 PDF 文档内嵌渲染与阅读体验。
- **AI 论文助手 (Papers AI)**：
  - 支持在导入 PDF 时自动解析并提取文档的元数据（标题、作者）。
  - 支持一键生成详细的论文结构化总结。
  - 支持针对当前阅读文献的上下文自由问答（RAG），AI 助手可以针对 PDF 文本进行精准答疑。
- **沉浸式 UI 重构 ("Ethereal Monolith" 风格)**：
  - 移除了老旧的星空粒子，引入了更具电影感和高级感的“体积流体光效（Volumetric Fluid）”。
  - 全局引入真实的胶片颗粒噪点（Film Grain）滤镜，提升界面质感。
  - 开屏动画与部分标题排版全面升级为具有超宽字间距的衬线字体（Serif Typography），营造深度思考的沉浸感。

## 🐛 修复与优化 (Fixes & Improvements)
- **修复**：Electron 渲染进程中调用原生 `window.prompt` 返回 `null` 导致的重命名操作静默失效问题，现已使用全局自定义 Dialog 替代。
- **修复**：在 Thoughts 视图中由于自动保存频繁修改时间戳，导致笔记列表发生不规则乱跳的问题（引入了脏数据检查机制）。
- **修复**：Windows 环境下由于拖拽区域 (`WebkitAppRegion`) 属性覆盖导致的文本框无法输入及光标跳转问题。
- **优化**：修复了在 Vite 环境下引入 `pdf-parse` (CommonJS 模块) 时的打包与运行兼容性问题。
- **优化**：优化了 AI 对话界面的“流式文档”视觉流向，将其从左右气泡修改为贴近现代阅读习惯的左侧对齐流布局，并将输入框下置。
